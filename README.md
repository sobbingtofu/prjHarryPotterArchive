# 해리포터 팬사이트
로고, 해리포터 API를 이용해 정보를 제공하고, 간단한 기숙사테스트 등을 통해 흥미를 유발 할 수 있는 해리포터 팬사이트를 제작했다
해리포터 영화나 등장인물 관련 정보를 알 수 있으며 리뷰 작성을 통해 다른 사용자들과 소통할 수 있다
